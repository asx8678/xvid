# Contributing

Thank you for considering contributing to X Video Downloader!

## How to contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Ensure tests still pass (`npm test`)
5. Submit a Pull Request

## Guidelines

- Keep permissions minimal
- Maintain test coverage
- Be defensive against X.com UI changes

For AI-assisted development, refer to `docs/AGENTS.md` and `docs/CLAUDE.md`.
